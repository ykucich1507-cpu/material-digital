# 🇦🇷 Guía Dropshipping Argentina con Dropi

Landing page de ventas + guía interactiva de 16 pasos para empezar a hacer dropshipping en Argentina con **Dropi + Tienda Nube + Mercado Pago**.

## 📁 Estructura

| Archivo | Descripción |
|---|---|
| `index.html` | Landing page de ventas (hero, beneficios, pricing, FAQ, CTA a Gumroad) |
| `guia.html` | El producto: guía interactiva de 16 pasos con checkboxes y barra de progreso. ⚠️ **No subir a hosting público** — es lo que se vende. |

## 🚀 Publicar la landing con GitHub Pages

1. Subí este repo a GitHub
2. Andá a **Settings → Pages**
3. En *Source* elegí `main` branch, carpeta `/ (root)`
4. Tu landing queda en `https://TU-USUARIO.github.io/guia-dropi-argentina/`

> 💡 Antes de publicar, considerá **eliminar o mover `guia.html`** fuera del repo público, ya que es el producto digital que se vende por Gumroad.

## 🛒 Venta
